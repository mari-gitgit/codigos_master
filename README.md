# Anexos Master Ciencia de Datos

Los presentes archivos y códigos fueron usados para el trabajo de grado titulado "Análisis y predicción de la Deserción en la Educación Superior Colombiana mediante Herramientas de Ciencia de Datos". 

Se encontrarán los códigos que se ejecutaron en máquina **local**, así como los archivos originales de donde se obtuvo la información.

Cada archivo de extracción representa el tratamiento de los diferentes archivos originales.

Encontramos también los archivos que se generan una vez se ejecutan los scripts originales en una máquina en la nube. Esto para el posterior procesamiento en el código que tendrá todos los procedimientos para ejecutar los modelos convenientes para este proyecto.

Con el fin de procesar los códigos en máquina local se deben descargar los archivos contenidos en la carpeta **archivos_fuente**.

Luego, vamos primero con los siguientes archivos, todos se encuentran en la carpeta **codigos_maquina_local**:\n

-escolaridad.py
-pobreza_monetaria.py
-pobreza_monetaria_sexo.py
-proyección_poblacion.py
-ruralidad.py
-scraping_estratos.py
-servicios_publicos.py
-tasas_empleo.py

Luego pasamos al EDA, primero se ejecuta:
-unificacion_df_EDA.py
Esto permite traer los archivos extraídos anteriormente unificados en uno solo.

Ahora sí el EDA:
-EDA.py

Las gráficas y resultados del análisis inicial se encuentran en la carpeta **EDA**.

Finalmente el modelo:
-Modelo.py

Los resultados del Modelo y las gráficas utilizadas se pueden ver en la carpeta **output_model**.
